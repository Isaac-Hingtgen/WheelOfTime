# WheelOfTime

Two player platformer mini game.

Goal: hit the other player downwards off the platform.

Controls:

p1:
A: move left
D: move right
Space: jump
W: normal attack
S: downwards attack (much larger hitbox)

p2: (requires keypad)
4: move left
6: move right
Enter: jump
8: normal attack
5: downwards attack (much larger hitbox)
